#Aplikacja mobilna o Harrym potterze.

Aplikacja używa API "https://hp-api.onrender.com/".

Ekran główny rozdziela aplikacje na podekrany postacie, zaklęcia oraz ulubione, które mają swoje podekrany z szczegółami.

Aplikacja posiada lokalną baze danych sqlite, shared_preferences, system dodawania do ulubionych oraz zakładke ustawień.
